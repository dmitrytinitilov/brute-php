# # Конструкторы

Добавим в наш класс конструктор

```php
class Number {

	public function __construct($num) {
		$this->num = $num;
	}
}
```


Теперь создание объекта будет выглядеть вот так

```php
$obj = new Number(5); //передаю параметр в конструктор
```
